Capítulo 2 – Estruturando Páginas Web com HTML

HTML (HyperText Markup Language) é uma das três linguagens fundamentais do desenvolvimento frontend, junto com CSS e JavaScript. Sua função é definir a estrutura das páginas web.

2.1 Introdução

2.1.1 Versões principais

  • HTML (1992) – Primeira versão, baseada em propostas de Tim Berners-Lee no CERN em 1989.
  • HTML 4.0 (1997) – Versão em que o CSS foi introduzido.
  • XHTML 1.0 – Reformulação do HTML com base em regras mais rígidas do XML.
  • HTML5 (2014) – Versão atual, com novas funcionalidades e APIs. Hoje mantida como "Living Standard" pelo W3C em colaboração com o WHATWG.

2.1.2 Elementos e Atributos

Cada elemento HTML possui uma tag de abertura e uma de fechamento. Os atributos ficam na tag de abertura e passam informações adicionais ao elemento.

Exemplo de elemento com atributo:

<a href="https://www.exemplo.com">Clique aqui</a>

2.1.3 Salvando Páginas como Documentos HTML

Páginas web são arquivos de texto simples com extensão .html ou .htm. Para visualizá-las, basta abrir o arquivo em um navegador. A maioria dos navegadores também oferece ferramentas de desenvolvedor para inspecionar o código HTML de qualquer página.

2.2 Principais Elementos

2.2.1 Títulos, Parágrafos e Formatação de Texto

O HTML oferece seis níveis de títulos: <h1> (maior) até <h6> (menor). Parágrafos são definidos com <p>. Para formatação, usa-se <b> (negrito), <i> (itálico), <em> (ênfase) e <strong> (forte). Quebras de linha são inseridas com <br />.

2.2.3 Links

Todos os tipos de links usam o elemento <a href="">. Há três categorias:

  • Links externos – apontam para outros sites (URL completa).
  • Links relativos – apontam para outras páginas do mesmo site (sem domínio).
  • Links internos – apontam para outra seção da mesma página, usando âncoras com #id.

Exemplo de link externo que abre em nova aba:

<a href="https://www.exemplo.com" target="_blank">Visitar site</a>

Boas práticas recomendam usar textos de link descritivos, como "Detalhes do produto", em vez de termos vazios como "Clique aqui".

2.2.4 Imagens

Imagens são incluídas com o elemento vazio <img>, usando os atributos src (caminho da imagem) e alt (texto alternativo). Desde o HTML5, é possível adicionar legenda usando <figure> e <figcaption>. 

<figure>
  <img src="imagens/foto.jpg" alt="Descrição da imagem" />
  <figcaption>Legenda da imagem</figcaption>
</figure>

2.2.5 Tabelas

Tabelas são definidas com <table>, linhas com <tr>, células com <td> e células de cabeçalho com <th>. Para tabelas longas, recomenda-se usar <thead>, <tbody> e <tfoot>. Células podem ser mescladas com colspan (colunas) e rowspan (linhas).

Exemplo de tabela simples de usuários:

Nome Sobrenome Cargo
Riana Fresh District Assurance Producer
Oscar Spielvogel Product Optimization Analyst
Lynn Berning Lead Accountability Administrator

2.2.6 Formulários

Formulários são criados com o elemento <form>. Os campos mais comuns incluem: campos de texto, senha, e-mail, radio buttons, checkboxes, listas dropdown (<select>), áreas de texto (<textarea>) e botão de envio. O elemento <label> adiciona rótulos acessíveis aos campos.

2.2.7 Informações Adicionais

  • Elementos de bloco – iniciam em nova linha (ex: <p>, <h1>, <ul>).
  • Elementos inline – ficam na mesma linha (ex: <a>, <b>, <img>).
  • Agrupamentos – <div> agrupa como bloco; <span> agrupa inline.
  • Caracteres de escape – ex: &lt; para < e &gt; para >.
  • Comentários – escritos com <!-- comentário -->.
  • Metadados – definidos com <meta> dentro do <head>.
  • Elementos estruturais (HTML5) – <header>, <footer>, <nav>, <article>, <aside>, <section>.

2.3 Resumo e Próximos Passos

Os pontos-chave aprendidos neste capítulo:

  • HTML é uma linguagem de marcação que define estrutura e semântica de páginas web.
  • Elementos HTML são compostos por tag de abertura e tag de fechamento.
  • Atributos passam informações adicionais à tag de abertura.
  • Alguns elementos são vazios e não contêm texto ou filhos.
  • Páginas HTML são arquivos de texto comuns.
  • Os elementos mais importantes incluem títulos, parágrafos, listas, links, imagens, tabelas e formulários.

O próximo passo é aprender CSS, a linguagem responsável por deixar as páginas visualmente atraentes.
